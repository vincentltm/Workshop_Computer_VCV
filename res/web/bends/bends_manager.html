Bends

Bends is a multi-stage lo-fi audio degradation processor, dynamic tape delay, granular glitcher, and ambient sample player. It takes incoming signals or internal ambient samples from clean through various stages of compression, dropouts, signal integrity failures, and resonant filtering.

Controls

  • Switch DOWN (Flick): Next Page
  • Switch DOWN (Hold, unadjusted): Previous Page
  • Switch DOWN (Hold + Tweak Main): Adjust global grittiness macro (bakes on release, locks page)
  • Switch UP (Flick): Toggle Latching Freeze
  • Switch UP (Hold, frozen): Open Page 7 (Freeze Scrub)
  • Switch UP (Hold, live): Open Page 6 (Sample Select/Player)
Page 0: Chorus
  • Main: Mix
  • X: LFO Rate
  • Y: Depth & FB
Page 1: Codec
  • Main: Mix (Strength)
  • X: Downsampling
  • Y: Ringing XOR
Page 2: Delay
  • Main: Mix
  • X: Delay Time
  • Y: Feedback
Page 3: Glitcher
  • Main: Mix
  • X: Loop Size
  • Y: Loop Speed
Page 4: Filter
  • Main: Cutoff
  • X: Res/Wavefold
  • Y: Morph
Page 5: Reverb
  • Main: Mix
  • X: Room Size
  • Y: Decay & Crunch

IO

  • Inputs
  • Audio 1: Main In (unplug to enable sample player)
  • Audio 2: Pitch CV (1V/Oct speed modulation on Page 6)
  • CV 1: Modulates Cutoff / Chorus / Delay Time
  • CV 2: Modulates Codec downsample / glitch mix
  • Pulse 1: Glitch trigger / Stutter repeat
  • Pulse 2: Freeze gate
  • Outputs
  • Audio 1/2: Stereo Output
  • CV 1: Visual LFO Phase / Envelope output
  • CV 2: Jitter / Random CV
  • Pulse 1: Clock Output
  • Pulse 2: Freeze State Output

Pages 6 & 7: Ambient Sample Player & Freeze Scrub

Page 6: Sample Player
  • Main Knob: Start position / scrubbing
  • X Knob: Playback Speed (Center 1.0x)
  • Y Knob: Loop Select (loads slots 1-28). Displays the selected slot temporarily in **1-based binary** on the 6 LEDs.
Page 7: Freeze Scrub
  • Main Knob: Scrub position inside the frozen granular buffer
  • X Knob: Playback Speed / Reverse mult
  • Y Knob: Grain loop length

Web Manager Interface

Importing Drag and drop .wav, .mp3 or other audio files. They are automatically normalized to -1dB and downsampled to the native 24kHz format.
Flashing Connect via USB, click Enter Flash Mode, then click Flash to Device to write your sample list directly to the card's flash memory.
Syncing Click Sync from Device to retrieve all current flash samples from the card to back up your current patches.
