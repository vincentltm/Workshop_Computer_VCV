Grains

Grains is a real-time granular processor. It fragments incoming audio into microscopic particles to reconstruct textures, clouds, and rhythmic patterns. It features sixteen internal memory slots for persistent audio storage and sample playback.

Controls

  • Down (Flick): Next Page
  • Down (Short): Previous Page
  • Up (Flick): Toggle Freeze / Record
  • Down (Hold 2s): Open Load/Save Menu
Page 1: Grains
  • Main: Position

  • Y: Size
Page 2: Tone
  • Main: Envelope
  • X: Jitter/Chord/Reverse
  • Y: Pitch
Page 3: Mix
  • Main: Wet/Dry
  • X: Feedback
  • Y: Spread
Page 4: Reverb

Web Manager Interface

Importing Drag and drop .wav or .mp3 files. They are automatically normalized and converted to the native 24kHz format.
Flashing Connect via USB and click Flash to Device to write your sample list directly to the card's flash memory.
Syncing Click Sync from Device to retrieve all 16 slots. This is required to back up on-module recordings.

On-Module Memory

The Save Menu
  1. In Freeze Mode (LED 6 ON), hold switch DOWN for 2s.
  2. LED 5 remains ON. Select slot 1-4 with Main Knob.
  3. Release to save the recording permanently to flash.
The Load Menu
  1. In Live Mode (LED 6 OFF), hold switch DOWN for 2s.
  2. LED 6 remains ON. Select slot 1-16 with Main Knob.
  3. Release to load the sample into the synthesis engine.
Slots 1-4: Persistent Recordings
Slots 5-16: User Flash Samples
LEDs 1-4 show Slot ID in 4-bit binary

Sample Player Engine

Tape Controls
  • Main Knob: Manual scrubbing and offset
  • X Knob: Playback Speed (Center 1.0x)
  • Y Knob: Pitch Control (Center 1.0x)
  • Switch UP: Pause playback
  • Switch DOWN (Hold): Open Slot Menu (1-16)
Tape IO
  • CV 1 In: 1V/Oct Pitch
  • CV 2 In: Position/Scrub
  • Pulse 1 In: Play/Pause
  • Pulse 2 In: Reset Playhead
  • Audio Out: Stereo Out
  • CV 1 Out: Playhead Ramp
  • CV 2 Out: Speed/Motion CV
  • Pulse 1 Out: End of Cycle
  • Pulse 2 Out: Midpoint Trigger
Activation: Hold DOWN during power-up. LED 5 indicates active mode. LEDs 0, 1, 3, 2 show progress.
